Cricsights 🏏📊

Cricsights is your all-in-one cricket intelligence platform—blending performance analytics with strategy and swagger.

✨ Features

  • Player search with detailed stats, strengths, and weaknesses
  • Batsman vs Bowler matchup analysis with graphs
  • Role-based comparison tools (openers, pacers, etc.)
  • Team Room: plan strategies, field placements, and notes
  • Fully responsive UI for fast insights before every match

🛠️ Tech Stack

Python, Flask, Pandas, Matplotlib, HTML/CSS, Tailwind, Cricket Data APIs
