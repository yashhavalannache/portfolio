Work Order System 📋⚙️

The Work Order System is a full-stack task management and tracking platform designed for streamlined collaboration between administrators and workers. It enables efficient task assignment, real-time status updates, proof-of-completion uploads, and direct communication — all within one responsive dashboard.

  • Task Assignment: Admins can create and assign tasks to specific workers with deadlines and priority levels.
  • Status Tracking: Workers can update task progress (Pending, In Progress, Completed) in real-time.
  • Proof Upload: Workers can attach images or files as proof of completion for admin verification.
  • Task-Based Chat: Real-time communication between admins and workers for clarifications and updates.
  • Worker Management: Admins can view registered workers, their profiles, and task history.

This system enhances productivity by keeping all task-related communication, documents, and progress tracking in one place. Whether for field work, maintenance teams, or internal task distribution — it ensures tasks are completed on time with clear accountability.

✨ Features

  • Role-based login for Admin and Worker
  • Admin dashboard with worker list and task overview
  • File upload support for proof of completion
  • Task-based chatbox for real-time communication
  • Responsive UI for desktop and mobile

🛠️ Tech Stack

Python, Flask, SQLite, HTML/CSS, Bootstrap/Tailwind, JavaScript
