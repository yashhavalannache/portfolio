SmartDoc Automator 🧠📄

Extract content from any media—text, audio, video, or images—then merge, summarize, and analyze like a pro.

✨ Features

  • Supports multiple file types: .txt, .pdf, .jpg, .mp3, .mp4
  • OCR (Tesseract), Speech-to-Text, and Video captioning
  • Extracts text, summarizes using NLP, generates .docx
  • Downloadable files with word count & AI summary
  • Attractive dark theme with stylish layout

🛠️ Tech Stack

Python, Flask, HTML, CSS, OpenCV, Tesseract OCR, MoviePy, HuggingFace Transformers
