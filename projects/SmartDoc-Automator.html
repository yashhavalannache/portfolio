SmartDoc Automator 🧠📄

SmartDoc Automator is a powerful AI-driven tool that transforms how you interact with digital content. Whether you're uploading a PDF, scanned image, audio clip, or even a video, SmartDoc intelligently extracts the embedded text using OCR, speech-to-text, or video transcription. Once extracted, it offers advanced features to merge multiple sources, generate AI-based summaries, calculate word counts, and export the final result in clean, downloadable formats — including .docx and Excel. It’s like having a digital assistant that reads, understands, and organizes your files — all in one sleek interface.

✨ Features

  • Supports multiple file types: .txt, .pdf, .jpg, .mp3, .mp4
  • OCR (Tesseract), Speech-to-Text, and Video captioning
  • Extracts text, summarizes using NLP, generates .docx
  • Downloadable files with word count & AI summary
  • Attractive dark theme with stylish layout

🛠️ Tech Stack

Python, Flask, HTML, CSS, OpenCV, Tesseract OCR, MoviePy, HuggingFace Transformers
