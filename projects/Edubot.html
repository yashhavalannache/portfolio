EduBot – Your Personal AI Learning Companion 🎓🤖

EduBot is an intelligent chatbot built to assist students with quick and relevant answers across academic topics. Powered by NLP and machine learning, it simulates a real-time Q&A assistant experience — offering definitions, examples, and humor, all from a lightweight local interface.

Designed for students by a student, EduBot uses a JSON-based intent-response model, enabling users to interact without needing an internet connection. It supports both text and voice input and is optimized for local deployment in classrooms, kiosks, or personal use.

✨ Features

  • Conversational chatbot with JSON-based intent handling
  • Voice recognition and text input support
  • Local deployment using Flask and Streamlit
  • Expandable intent-response model for various subjects
  • Minimalist UI with quick response generation

🛠️ Tech Stack

Python, Flask, Streamlit, SpeechRecognition, JSON, HTML/CSS
